<h1 align="center">🛒 GameSpace</h1>

<p align="center">
Full-stack e-commerce web application built with <b>Vue 3</b> and <b>PHP 8.2</b>
</p>

<p align="center">
GameSpace is a full-stack e-commerce web application built with Vue 3 on the frontend and PHP 8.2 on the backend.
The project demonstrates how a modern reactive frontend communicates with a backend API to implement common online store functionality.
</p>

<p align="center">
Users can browse products, search and filter items, manage a shopping cart, simulate checkout, and view their order history.
The application also includes an admin interface for managing products and monitoring store data.
</p>

<hr>

<h2>Features</h2>

<h3>Store Functionality</h3>
<ul>
<li>Product listing</li>
<li>Live product search</li>
<li>Product filtering</li>
<li>Product pagination</li>
<li>Product detail pages</li>
</ul>

<h3>Shopping System</h3>
<ul>
<li>Shopping cart</li>
<li>Checkout simulation</li>
<li>Order creation stored in the database</li>
</ul>

<h3>User System</h3>
<ul>
<li>User registration</li>
<li>User login</li>
<li>User profile settings</li>
<li>Editing user information</li>
<li>Order history</li>
</ul>

<h3>Admin Panel</h3>
<ul>
<li>Product management (create / edit / update)</li>
<li>Orders slideshow</li>
<li>Simple dashboard with KPI metrics</li>
</ul>

<hr>

<h2>Tech Stack</h2>

<h3>Frontend</h3>
<ul>
<li>Vue 3</li>
<li>Vite</li>
<li>JavaScript</li>
<li>CSS</li>
</ul>

<h3>Backend</h3>
<ul>
<li>PHP 8.2</li>
<li>MySQL database</li>
</ul>

<h2>Installation</h2>

<pre>
git clone https://github.com/adamsima77/GameSpace
cd GameSpace
npm install
npm run dev
</pre>

<hr>

<h2>Backend Setup</h2>

<ol>
<li>Place the project in a local server environment (XAMPP / WAMP / MAMP)</li>
<li>Configure the database connection</li>
<li>Import the MySQL database</li>
<li>Run the PHP backend</li>
</ol>
